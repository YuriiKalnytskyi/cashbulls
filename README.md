## Getting Started

#### Below are two examples of how to run this project :

### The first way:

```bash
$ cd server
$ npn install
$ npm run db:start:local_server 
```

##### And open a new terminal window:

```bash
$ cd client
$ npn install
$ npm run db:start:local_server 
```
